## このプラグインで出来ること

* キーボード入力を受け付けます

!!! Info "こういう時に使おう"
    * 翻訳の調整や「どうしても認識しない文字を入れる」ときに使いましょう

##　有効化

![direct](images/plugin_directinput_p1.png)

* プラグインを使うチェックをONにしてください。

## 設定

![direct](images/plugin_directinput_p2.png)

* ウィンドウが出ます。
* ウィンドウサイズに伴って文字サイズなども大きくなります
* 文字を入力し、エンターもしくは送信で文字をおくることができます
